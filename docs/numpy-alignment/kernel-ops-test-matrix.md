## 1. ArgMax/ArgMin with Axis

### 1.1 Basic 2D Array Tests

```python
arr2d = np.array([[1, 5, 2], [4, 3, 6]])  # shape (2, 3)
```

| Operation | Result | Notes |
|-----------|--------|-------|
| `np.argmax(arr2d)` | `5` | Flattened index of max value (6) |
| `np.argmax(arr2d, axis=0)` | `[1 0 1]` | Index of max along rows |
| `np.argmax(arr2d, axis=1)` | `[1 2]` | Index of max along cols |
| `np.argmax(arr2d, axis=-1)` | `[1 2]` | Same as axis=1 |
| `np.argmax(arr2d, axis=-2)` | `[1 0 1]` | Same as axis=0 |
| `np.argmin(arr2d)` | `0` | Flattened index of min value (1) |
| `np.argmin(arr2d, axis=0)` | `[0 1 0]` | Index of min along rows |
| `np.argmin(arr2d, axis=1)` | `[0 1]` | Index of min along cols |
| `np.argmin(arr2d, axis=-1)` | `[0 1]` | Same as axis=1 |
| `np.argmin(arr2d, axis=-2)` | `[0 1 0]` | Same as axis=0 |

### 1.3 NaN Handling

**CRITICAL**: NaN propagates in comparisons - argmax/argmin returns index of first NaN encountered.

```python
arr_nan = np.array([[1.0, np.nan, 3.0], [4.0, 5.0, 6.0]])
```

| Operation | Result | Notes |
|-----------|--------|-------|
| `np.argmax(arr_nan)` | `1` | Index of NaN (flattened) |
| `np.argmax(arr_nan, axis=0)` | `[1 0 1]` | NaN at [0,1] wins axis=0 comparison |
| `np.argmax(arr_nan, axis=1)` | `[1 2]` | Row 0: NaN at index 1; Row 1: 6 at index 2 |
| `np.argmin(arr_nan)` | `1` | NaN also wins min comparison |
| `np.argmin(arr_nan, axis=0)` | `[0 0 0]` | NaN at [0,1] wins |
| `np.argmin(arr_nan, axis=1)` | `[1 0]` | Row 0: NaN wins |

**All NaN in reduction axis:**
```python
arr_all_nan = np.array([[np.nan, np.nan], [1.0, 2.0]])
np.argmax(arr_all_nan, axis=1)  # [0 1] - first NaN index for row 0
np.argmin(arr_all_nan, axis=1)  # [0 0] - first NaN index for row 0
```

## 2. All/Any

### 2.1 Basic Boolean Tests

```python
arr2d = np.array([[True, False, True], [True, True, True]])  # shape (2, 3)
```

| Operation | Result | Notes |
|-----------|--------|-------|
| `np.all(arr2d)` | `False` | One False exists |
| `np.all(arr2d, axis=0)` | `[True False True]` | Column 1 has False |
| `np.all(arr2d, axis=1)` | `[False True]` | Row 0 has False |
| `np.all(arr2d, axis=-1)` | `[False True]` | Same as axis=1 |
| `np.any(arr2d)` | `True` | At least one True |
| `np.any(arr2d, axis=0)` | `[True True True]` | All columns have True |
| `np.any(arr2d, axis=1)` | `[True True]` | All rows have True |

### 2.2 Numeric Truthiness

**Rule**: 0 is False, all non-zero (including NaN) is True.

```python
arr_int = np.array([[0, 1, 2], [3, 0, 5]])
```

| Operation | Result | Notes |
|-----------|--------|-------|
| `np.all(arr_int)` | `False` | Contains zeros |
| `np.all(arr_int, axis=0)` | `[False False True]` | Col 0,1 have zeros |
| `np.all(arr_int, axis=1)` | `[False False]` | Both rows have zeros |
| `np.any(arr_int)` | `True` | Has non-zero values |
| `np.any(arr_int, axis=0)` | `[True True True]` | All cols have non-zero |
| `np.any(arr_int, axis=1)` | `[True True]` | All rows have non-zero |

### 2.3 Empty Arrays

**CRITICAL**: Empty all() returns True (vacuous truth), empty any() returns False.

| Array Shape | `np.all()` | `np.any()` | Notes |
|-------------|------------|------------|-------|
| `(0,)` | `True` | `False` | Vacuous truth / no True exists |
| `(0, 3)` axis=0 | `[True True True]` | `[False False False]` | Per-column vacuous truth |
| `(0, 3)` axis=1 | `[]` | `[]` | No rows to reduce |
| `(2, 0)` axis=0 | `[]` | `[]` | No columns to reduce |
| `(2, 0)` axis=1 | `[True True]` | `[False False]` | Per-row vacuous truth |

### 2.4 NaN Handling

**CRITICAL**: NaN is truthy (non-zero)!

```python
arr_nan = np.array([0.0, np.nan, 1.0])
np.all(arr_nan)   # False - 0.0 is falsy
np.any(arr_nan)   # True  - NaN and 1.0 are truthy

arr_nan_only = np.array([np.nan, np.nan])
np.all(arr_nan_only)  # True  - NaN is truthy
np.any(arr_nan_only)  # True  - NaN is truthy
```

| Array | `np.all(axis=0)` | `np.all(axis=1)` | `np.any(axis=0)` | `np.any(axis=1)` |
|-------|------------------|------------------|------------------|------------------|
| `[[0, nan], [nan, 0]]` | `[False False]` | `[False False]` | `[True True]` | `[True True]` |

## 3. Prod (Product)

### 3.1 Basic Tests

```python
arr2d = np.array([[1, 2, 3], [4, 5, 6]])  # shape (2, 3)
```

| Operation | Result | Notes |
|-----------|--------|-------|
| `np.prod(arr2d)` | `720` | 1*2*3*4*5*6 |
| `np.prod(arr2d, axis=0)` | `[4 10 18]` | Column products |
| `np.prod(arr2d, axis=1)` | `[6 120]` | Row products |
| `np.prod(arr2d, axis=-1)` | `[6 120]` | Same as axis=1 |
| `np.prod(arr2d, axis=-2)` | `[4 10 18]` | Same as axis=0 |

### 3.2 Empty Arrays

**CRITICAL**: Empty prod returns 1 (multiplicative identity).

| Array Shape | `np.prod()` | Notes |
|-------------|-------------|-------|
| `(0,)` | `1` | Identity element |
| `(0, 3)` axis=0 | `[1 1 1]` | Per-column identity |
| `(0, 3)` axis=1 | `[]` | No rows to reduce |
| `(2, 0)` axis=0 | `[]` | No columns to reduce |
| `(2, 0)` axis=1 | `[1 1]` | Per-row identity |

### 3.3 NaN Handling

```python
arr_nan = np.array([1.0, np.nan, 3.0])
np.prod(arr_nan)      # nan
np.nanprod(arr_nan)   # 3.0  (ignores NaN)
```

| Array | `np.prod(axis=0)` | `np.prod(axis=1)` |
|-------|-------------------|-------------------|
| `[[1, nan], [2, 3]]` | `[2 nan]` | `[nan 6]` |

### 3.4 Overflow Behavior

**CRITICAL**: Integer overflow wraps silently, float overflow returns inf.

| Array | dtype | `np.prod()` | Notes |
|-------|-------|-------------|-------|
| `[10^15, 10^15, 10^15]` | int64 | `802379605485813760` | Silent overflow/wrap |
| `[100, 100, 100]` | uint8 | `1000000` (uint64) | Promoted to uint64 |
| `[1e200, 1e200, 1e200]` | float64 | `inf` | Float overflow |
| `[1e-200, 1e-200, 1e-200]` | float64 | `0.0` | Float underflow |

### 3.5 Zero Handling

```python
arr_zero = np.array([1, 2, 0, 4, 5])
np.prod(arr_zero)  # 0

arr_zero_2d = np.array([[1, 0, 3], [4, 5, 6]])
np.prod(arr_zero_2d, axis=0)  # [4 0 18]
np.prod(arr_zero_2d, axis=1)  # [0 120]
```
