## What's Changed
* Enabled NDArray boolean comparisons for LessThan, GreaterThan, and … by @Rikki-Tavi in https://github.com/SciSharp/NumSharp/pull/395
* Added data types in np.frombuffer. in https://github.com/SciSharp/NumSharp/pull/425
* F# in README by @dsyme in https://github.com/SciSharp/NumSharp/pull/432
* Added support for user defined decimal precision for np.around() and TensorEngine.Round() by @shashi4u in https://github.com/SciSharp/NumSharp/pull/453
* NumSharp.Bitmap support for odd sized bitmaps with odd sized bytes per pixel by @AmbachtIT in https://github.com/SciSharp/NumSharp/pull/460
* Fixing the consistency of seed in the random choice. by @bojake in https://github.com/SciSharp/NumSharp/pull/489
* <feat>(Logics):add high performance logical AND function with axis an… by @zhuoshui-AI in https://github.com/SciSharp/NumSharp/pull/525
* Upgrade target frameworks to net8.0;net10.0 by @Nucs in https://github.com/SciSharp/NumSharp/pull/532
* Add GitHub Actions CI/CD pipeline by @Nucs in https://github.com/SciSharp/NumSharp/pull/534
* Fix: skip Bitmap tests on non-Windows CI by @Nucs in https://github.com/SciSharp/NumSharp/pull/535
* docs: relocate website to docs/website/ by @Nucs in https://github.com/SciSharp/NumSharp/pull/557
* docs: move docfx_project to docs/website-src by @Nucs in https://github.com/SciSharp/NumSharp/pull/558
